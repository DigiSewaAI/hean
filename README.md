# 🏨 HEAN - Hostel Entrepreneur Association of Nepal

HEAN is a complete web application for managing hostel registrations, committee members, notices, gallery, and certificates for the Hostel Entrepreneur Association of Nepal. It features a bilingual (EN/NE) public website and a comprehensive admin dashboard with full CMS capabilities.

---

## 🚀 Features

- **Bilingual Public Website** (English / Nepali)
- **Admin Dashboard** with full CMS
- **Hostel Registration & Management**
- **Committee Member Management**
- **Notice & Event Management**
- **Gallery Management**
- **Certificate Generation**
- **Inspection Checklist**
- **Reports & Statistics**
- **Mobile Responsive Design**

---

## 📋 Requirements

- PHP ≥ 8.2
- Composer
- Node.js (for Tailwind/Assets)
- MySQL
- Laragon (for Windows) or any local server
